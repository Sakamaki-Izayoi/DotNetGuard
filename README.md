# DotNetGuard
.Net Protector

### Features
    * Anti-Debug
    * Anti-Dump
    * Anti-ILDasm
    * Arithmetic
    * Control flow
    * Control flow 2
    * Drag & drop
    * KoiVM
    * Pack
    * Rename
    * Signer
    * String Encryption
