# [DotNetGuard](https://github.com/Modify24x7/DotNetGuard/)
.Net Protector

### Features
    * Anti-Debug
    * Anti-Dump
    * Anti-ILDasm
    * Arithmetic
    * Control flow
    * Control flow 2
    * Drag & drop
    * KoiVM Light Mod
    * Pack
    * Rename
    * Signer
    * String Encryption
